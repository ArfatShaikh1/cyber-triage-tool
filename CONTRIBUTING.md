# Contributing to Cyber Triage Tool

Welcome! We are excited that you want to contribute to Cyber Triage Tool. Your help is crucial for the growth of this project. Here are some guidelines to get you started:

## How to Contribute
1. **Fork the repository** - Start by creating a personal fork of the project.
2. **Clone your fork** - Clone the repository to your local machine:
   ```
   git clone https://github.com/YOUR_USERNAME/cyber-triage-tool.git
   ```
3. **Create a new branch** - Make sure to create a new branch for your feature or fix:
   ```
   git checkout -b feature-name
   ```
4. **Make your changes** - Implement your feature or bug fixes.
5. **Test your changes** - Run tests to ensure your changes don't break anything.
6. **Commit your changes** - Commit your changes with a clear message:
   ```
   git commit -m "Add your message"
   ```
7. **Push to your fork** - Push your branch to your fork:
   ```
   git push origin feature-name
   ```
8. **Open a pull request** - Go to the original repository and open a pull request.

## Guidelines
- Make sure your code adheres to the existing coding style. 
- Write clear and concise commit messages.
- Update documentation as needed.
- Ensure your changes do not break existing functionality.
